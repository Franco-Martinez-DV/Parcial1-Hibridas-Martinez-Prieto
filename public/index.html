ARTISTS API

En la era digital actual, la música es una de las formas de arte más accesibles y apreciadas en todo el mundo. Con el propósito de facilitar la interacción y el descubrimiento de artistas y sus álbumes, diseñamos una API intuitiva y eficiente que permite a desarrolladores y entusiastas acceder a una amplia base de datos de música. Esta API ofrece funcionalidades para explorar información detallada sobre artistas, sus álbumes, géneros, fechas de lanzamiento y más, promoviendo una experiencia enriquecedora que conecta a los usuarios con su música favorita. A través de esta plataforma, buscamos potenciar la creatividad y la innovación en aplicaciones relacionadas con la música, brindando las herramientas necesarias para construir proyectos que celebren el talento musical global.

ARTISTS

Utiliza nuestra API para acceder a todos los artistas disponibles en nuestra base de datos.

get
http://127.0.0.1:3000/api/artists/
Obtiene todos los artistas.
post
http://127.0.0.1:3000/api/artists/
Permite agregar artistas.
get
http://127.0.0.1:3000/api/artists/:id
Obtiene los artistas por ID.
put
http://127.0.0.1:3000/api/artists/:id
Actualiza los artistas por ID.
delete
http://127.0.0.1:3000/api/artists/:id
Borra los artistas por ID.
get
http://127.0.0.1:3000/api/artists/name/:name
Obtiene los artistas por nombre artístico.
get
http://127.0.0.1:3000/api/artists/genre/:genre
Obtiene los artistas por género musical.
get
http://127.0.0.1:3000/api/artists/country/:country
Obtiene los artistas por país de origen.
